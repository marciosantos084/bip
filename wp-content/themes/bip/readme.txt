== Description ==

BIP é um tema WordPress exclusivo.

== Installation ==

1. No painel de administração, vá para Aparência > Temas e clique no botão Adicionar novo.
2. Clique em Carregar e Escolher Arquivo e selecione o arquivo .zip do tema. Clique em Instalar agora.
3. Clique em Ativar para usar seu novo tema imediatamente.

= Theme Features Usage =
Todas as opções disponíveis podem ser usadas em Aparência->Personalizar
